**Weather Widget - Installation Usage**

# Setup

Get API Key from OpenWeatherMap

# Create .env file:

```env
OPENWEATHER_API_KEY=your_api_key_here
```

# Development

```sh
npm run dev
```

# Production Build

```sh
npm run build
```

Creates dist/weather-widget.js

# Embedding

Add to any HTML page:

```html
<weather-widget />
<script
    type="text/javascript"
    src="https://cdn.jsdelivr.net/gh/VaryaEr/test-widget@main/dist/weather-widget.js"
></script>
```
